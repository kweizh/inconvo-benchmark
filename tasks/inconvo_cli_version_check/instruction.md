# Inconvo CLI Version Check

## Background
You have the Inconvo CLI installed in the environment. Your task is to check the version of the installed CLI.

## Requirements
- Check the installed version of the `inconvo` CLI.
- Save the version output to a file named `version.txt` in the project directory.

## Implementation Guide
1. Run the command to get the `inconvo` version (usually `inconvo --version` or `inconvo -V`).
2. Redirect the output to `/home/user/project/version.txt`.

## Constraints
- Project path: /home/user/project
- Log file: /home/user/project/version.txt
